
# 在线聊天室

*******************
环信MQTT参赛作品

## 运行环境
iOS 9.0+

## 功能简介
构建三个聊天室，每台设备对应一个用户，可以在聊天室内畅所欲言(基于环信MQTT).

## 资源说明
图标，ICON，封面等资源均为开发使用，并非商业用途。
